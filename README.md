# LightFlow-ACR

A lightweight real-time multivariate time series anomaly detection, classification and correction system.

## Features

* Real-time anomaly detection
* RRCF-based streaming analysis
* Anomaly classification
* Data correction
* Online monitoring

## Tech Stack

* Python
* RRCF
* LSTM
* Docker
* Grafana

## Dataset

* FEMTO-ST
* NASA CMAPSS
